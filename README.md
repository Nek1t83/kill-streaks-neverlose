# Kill Streaks (Neverlose CS:GO)

Lua-скрипт для чита **Neverlose** в CS:GO. Проигрывает звук и показывает баннер
в центре экрана за каждый килл в раунде:

| Киллов в раунде | Стрик       | Баннер           |
|-----------------|-------------|------------------|
| 1               | First Blood | `first_blood.png` |
| 2               | Double Kill | `double_kill.png` |
| 3               | Triple Kill | `triple_kill.png` |
| 4               | Quadra Kill | `dominating.png`  |
| 5+              | Penta Kill  | `rampage.png`     |

Счётчик сбрасывается в начале каждого раунда. Тиммейтов и суициды по умолчанию
не считаем (тиммейтов можно включить в меню).

Дополнительно есть **авто-ресет КД** — после смерти скрипт смотрит счёт и если
смертей больше киллов, шлёт в чат `!rs` (команда сервера на ресет статистики).

---

## Что в репозитории

```
kill_streaks.lua    — сам скрипт
images/             — PNG-баннеры стриков (5 шт.)
sounds/             — сюда нужно положить ваши WAV-звуки (см. ниже)
```

Звуки в репозиторий не залиты (нет прав на распространение оригиналов из
Unreal Tournament / Quake). Их легко найти в гугле по запросу
`unreal tournament announcer first blood wav` и т.п.

## Установка

### 1. Скрипт

1. Открой Neverlose → **LUA** → **Add script**.
2. Выбери файл `kill_streaks.lua` из этого репозитория.
3. Включи скрипт галочкой.

В меню чита появится секция **Kill Streaks** с настройками:

- **Включить стрики** — главный переключатель.
- **Учитывать тиммейтов** — считать ли убийства союзников.
- **Громкость** — 0..200% (50% = громкость как у обычного `play` в консоли,
  выше — складываем несколько копий звука).
- **Картинки стриков** — показывать ли баннер по центру экрана.
- **Размер картинок** — 50..200%, базовый размер ~18% от высоты экрана.
- **Авто-ресет КД** — слать `!rs` в чат после смерти, если K < D.

### 2. Звуки (`sounds/`)

Тебе нужны 5 файлов в формате **WAV**:

```
first_blood.wav
double_kill.wav
triple_kill.wav
quadra_kill.wav
penta_kill.wav
```

Положи их в свою папку CS:GO по пути:

```
…\Steam\steamapps\common\Counter-Strike Global Offensive\csgo\sound\streaks\
```

Папку `streaks` создай вручную, если её нет. После этого скрипт сможет
вызывать `playvol streaks/first_blood.wav` через консоль, и звук
проиграется.

**Если звука нет** — путь будет пустой, скрипт не упадёт, просто ничего не
сыграет.

**Формат:** PCM WAV, 22050 или 44100 Hz, моно или стерео. `.mp3` и `.ogg`
CS:GO не любит — конвертируй в WAV (Audacity / ffmpeg).

### 3. Картинки (`images/`)

В репозитории уже лежат 5 готовых PNG-баннеров (размер 1280×512, соотношение
2.5:1). Положи их в папку Neverlose со скриптами:

```
%localappdata%\neverlose\
```

Точнее, скрипт грузит файлы по пути `streaks/first_blood.png` через
`render.load_image_from_file`. Это означает, что **картинки должны лежать
в той же папке, откуда Neverlose грузит сам Lua-скрипт**, во вложенной папке
`streaks`. Например:

```
%localappdata%\neverlose\scripts\streaks\first_blood.png
%localappdata%\neverlose\scripts\streaks\double_kill.png
…
```

Если картинки не загрузятся — увидишь стрик без баннера, звук всё равно
сыграет. Скрипт кэширует результат `pcall(render.load_image_from_file, …)`
и больше не пытается их грузить, так что лагать не будет.

Свои картинки можно подсунуть с теми же именами — главное соотношение сторон
**~2.5:1**, иначе будет растянуто. Под ультравайд скрипт сам ограничивает
ширину баннера (привязка к высоте экрана).

## FAQ

**Звуки не играют.**
Проверь путь: `csgo/sound/streaks/first_blood.wav`. Открой консоль CS:GO и
выполни `playvol streaks/first_blood.wav 1` — если играет вручную, значит
скрипт тоже сыграет.

**Картинки не показываются.**
Скрипт грузит их через `render.load_image_from_file('streaks/first_blood.png', …)`.
Путь относительный — относительно рабочей папки Neverlose. Если не уверен,
куда класть, попробуй сначала в одну папку со скриптом, потом во вложенную
`streaks/`.

**Баннер мелкий / огромный.**
Крути слайдер «Размер картинок» (50..200%). Базовый размер = 18% от высоты
экрана.

**На ультравайде баннер не растягивается на весь экран.**
Так и задумано — ширина баннера привязана к высоте экрана, не ширине.

**Авто-ресет шлёт `!rs` в команду, но сервер не понимает.**
`!rs` работает на community-серверах с плагином SourceMod. На матчмейкинге
и на чистых серверах — никак, скрипт просто напишет фразу в чат.
